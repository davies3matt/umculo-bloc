## Tools

- Monorepo managed by [lerna](https://github.com/lerna/lerna)
- Next.js web3 starter package in `/packages/next-app` - [[source repo]](https://github.com/mirshko/next-web3-boilerplate)

### Bootstrap dev env

install root dependencies

```bash
# /
npm install
```

bootstrap packages

```bash
# /
npx lerna bootstrap
```
